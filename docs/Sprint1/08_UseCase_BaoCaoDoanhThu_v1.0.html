II. ĐẶC TẢ CHI TIẾT USE CASE — UC-BC03
Trường thông tinNội dung chi tiết
Tên Use CaseXuất báo cáo dạng Excel
Mã sốUC-BC03
Tác nhânNhân viên Kế toán, Quản lý
Mô tảKết xuất dữ liệu đang hiển thị trên lưới Pivot Grid ra file định dạng Excel (.xlsx) để phục vụ lưu trữ, in ấn hoặc phân tích chuyên sâu.
Tiền điều kiệnNgười dùng đã thực hiện truy vấn (UC-BC01) và dữ liệu đang hiển thị trên màn hình.
Luồng sự kiện chính
  1. Người dùng nhấn chọn chức năng "Xuất Excel" trên thanh công cụ của báo cáo.
  2. Hệ thống hiển thị hộp thoại "Save File Dialog" để người dùng chọn vị trí lưu và đặt tên file.
  3. Người dùng xác nhận vị trí lưu.
  4. Hệ thống thực hiện chuyển đổi cấu trúc dữ liệu Pivot (giữ nguyên các cột/dòng đang mở rộng) sang định dạng Excel.
  5. Hệ thống thông báo "Xuất báo cáo thành công" và cung cấp tùy chọn mở file ngay lập tức.
Ngoại lệLỗi truy cập file (đĩa đầy, file đang mở bởi ứng dụng khác): Hệ thống hiển thị thông báo lỗi và yêu cầu người dùng kiểm tra lại.
Hậu điều kiệnMột file .xlsx được tạo ra chứa toàn bộ dữ liệu như hiển thị trên giao diện tại thời điểm xuất.
